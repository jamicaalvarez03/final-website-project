# Deployment Steps

## 1. Upload to GitHub
Create a new GitHub repository and upload:
- `index.html`
- `styles.css`
- `script.js`
- `README.md`

Commit your files.

## 2. Enable GitHub Pages
1. Open the repository
2. Click **Settings**
3. Click **Pages**
4. Under **Source**, choose **Deploy from a branch**
5. Select the `main` branch and `/root`
6. Save and wait for publishing

## 3. Copy Your Links
After publishing, copy:
- the live website link
- the GitHub repository link

## 4. Submit
Paste both into your course submission.
