About

We created a café experience that feels warm and welcoming.

GreenSprout Café is built around comfort, quality, and community. Customers can quickly learn what we offer, browse the menu, and contact us through a simple, mobile-friendly layout.

This project demonstrates responsive design, accessible structure, semantic HTML, and clean CSS organization.

Why this project works well

  • Easy navigation for first-time visitors

  • Professional structure for GitHub review
